After more than a decade . . . Version 2!

This is the (new) support site for my JavaScript Event Calendar. Here, you’ll find information about how to use the calendar, including lots of examples. The JavaScript Event Calendar began in the Fall of 1998 (!!!) and was rewritten in 2010 from the ground up, making it easier for webmasters to customize the look-and-feel and to define events. The sidebar above highlights the new features; be sure to check out the Tutorial page (to understand the run-time options and CSS settings) and the Examples (to appreciate the variety of calendars that you can create). You can download the script and some sample CSS files, so you’ll be up and running in a matter of minutes!
